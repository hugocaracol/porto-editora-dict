porto-editora-dict
==================

Portuguese dictionary

Dependencies:
* BeautifulSoup4
* urllib2

Usage:
<pre>
<code>./porto-dict &lt;word&gt;</code>
</pre>

<pre><code>
hugos@caracol:~/sandbox/git/porto-editora-dict$ ./porto-dict.py olá
olá [ó] (interjeição)
1. usada como saudação ou chamamento
2. exprime espanto
</code></pre>
